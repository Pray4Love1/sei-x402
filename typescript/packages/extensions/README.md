# x402 Extensions

This package provides utilities and examples for facilitator fee extensions.

## Quick Start

```bash
cd typescript/packages/extensions
npm install
npm run sign-quote
npm run verify-quote -- --signature <signature> --facilitator-address <address>
npm run payment-required
```
